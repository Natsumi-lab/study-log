# モジュールとは  
モジュールとは「Python のコードをまとめたファイル」のこと。  
Python では、ファイル（.py）ごとに機能を分けて書き、それらを import して使います。  

```  
import モジュール名
import モジュール名 as 別名    
```  
asを付けるとモジュール名を短く書けて便利になる。  
例：  
- numpy → np  
- pandas → pd  
- matplotlib.pyplot → plt  

## モジュールの特徴
1. Python で利用できる「機能のまとまり」  
計算、文字の操作、ファイル処理などをまとめたもの。  

2. 自分で作ることもできる  
mymodule.py のようにファイルを作れば、それがモジュールになります。  

3. import して使う  
必要なモジュールを読み込んで使う仕組み。  

## Python モジュールの種類  

### 🍡標準モジュール（Python に最初から入っているもの）  
インストール不要で、すぐに使えるモジュール  

| モジュール          | 用途                   |
| -------------- | -------------------- |
| **math**       | 数学計算（平方根、三角関数など）     |
| **random**     | 乱数を作る                |
| **datetime**   | 日付や時刻を扱う             |
| **os**         | OS（フォルダ操作・環境変数など）と連携 |
| **sys**        | Python自体の動作を制御       |
| **json**       | JSONデータの読み書き         |
| **re**         | 正規表現（文字列の検索・置換）      |
| **statistics** | 統計（平均・中央値など）         |

  
### 🍡外部モジュール（自分でインストールするもの）  
pip install でインストールして使うパッケージ  

| モジュール              | 分野      | 用途          |
| ------------------ | ------- | ----------- |
| **numpy**          | 数値計算    | 高速な計算処理     |
| **pandas**         | データ分析   | 表形式データの処理   |
| **matplotlib**     | グラフ描画   | グラフ作成       |
| **requests**       | Webアクセス | HTTP通信      |
| **Flask / Django** | Web開発   | Webアプリ作成    |
| **opencv-python**  | 画像処理    | 画像認識・編集     |
| **pyautogui**      | 自動化     | マウス・キーボード操作 |
