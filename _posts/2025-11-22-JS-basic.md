# 基本構文
## if文      
条件によって 処理を分ける ための基本構文
```
if (条件式) {
  // 条件が true のときの処理
}
```

## else節      
if の条件が false のとき に実行する処理を書く部分  
```
if (条件式) {
  // trueの場合
} else {
  // falseの場合
}
```

## alert()      
画面にポップアップでメッセージを表示する（OKボタンだけの小さいダイアログ）
```
alert('送信が完了しました！');
```
